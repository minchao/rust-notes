# 工具

## 開發工具

### rustfmt

https://github.com/rust-lang/rustfmt

程式碼格式化工具，會根據 Rust 程式碼風格指南進行格式化。

安裝：

```shell
$ rustup component add rustfmt
```

執行：

```shell
$ rustfmt [options] <file>...
```

或在使用 Cargo 管理的專案下執行：

```shell
$ cargo fmt
```
